HomeLab

Dans le cadre d'un projet personnel, j'ai conçu et déployé un serveur Unraid puissant et sécurisé. Cette configuration utilise des technologies modernes pour optimiser l'accès aux services tout en garantissant la protection des données.

Caractéristiques Principales :

  • Gestion des Accès : Utilisation de Cloudflare en tant que proxy pour sécuriser les connexions Internet, avec un accès via Wireguard pour les outils d'administration.
  • Containerisation avec Docker : Hébergement de diverses applications, incluant Jellyfin pour le streaming et Vaultwarden pour la gestion des mots de passe.
  • Support de Base de Données : Intégration de PostgreSQL pour la gestion des données et de Redis pour le cache, améliorant ainsi la performance des applications.
  • Sécurité Renforcée : Implémentation de CrowSec pour une protection proactive contre les menaces, et de Traefik pour la gestion flexible des requêtes.
